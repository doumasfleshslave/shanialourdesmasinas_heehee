LESSON 1: EVENTS & POP-UP BOXES

Events

  • Actions that are triggered either by mouse clicks, function calls, keyboard strokes, loading of images, submitting an HTML form, etc.
  • Usually used in connections with functions

Event Handlers

  • They help call functions at the right time, ensuring they function properly and respond to the page's events.

Event Triggers

Attribute Description

Pop-up Boxes

  • Windows that appear before a page loads, events, or page exits.
  • Serve as warnings, prompts, or confirmation boxes.

LESSON 2: CONDITIONS, CONTROLS, LOOPS

  • Conditional Statements perform different actions for different decisions.

IF Statement

  • Used to check or verify a condition.
  • Executes statements only if the condition is true.

Nested IF Statement

  • "If statement" with another "If statement"
  • Used when the child condition is checked only if the parent condition is true

IF-ELSE Statement

  • Adds an alternative instruction if the condition is not true.

IF ELSE IF Statement

  • Evaluates multiple conditions.
  • Executes different blocks depending on which condition is true.

Switch Statement

  • Used when a condition has multiple results.
  • A different set of operations is executed depending on the result.

LESSON 3: FUNCTIONS

  • A function is a self-contained piece of code that performs a specific task.
  • Also called a method.
  • Used so you do not have to retype your code again.
  • Can be called anywhere in the page or from an external JavaScript file.
  • Usually placed in the head section to ensure it loads properly.
  • Defined using the keyword function.

Return Statement

  • Used when a function needs to return a value.
  • Allows the output of a function to be used elsewhere.

LESSON 4: LOOPS

  • A repetitive cycle of code until a condition is met.

Types of Loops

  • For Loop – repeats code a specific number of times.
  • While Loop – repeats while the condition remains true.
  • Do-While Loop – runs the code once before checking the condition.
  • For-In Loop – loops through elements of an array or object.

Break Statement

  • Stops the loop when the specified condition becomes true.

Continue Statement

  • Skips a value but continues the loop.

LESSON 5: STRINGS AND ARRAY OBJECTS, DATE, TIME AND MATH OBJECTS

String

  • A string is an object with its own properties and methods.
  • Used to manipulate text.
var text="This is a new house."; document.write(text.length);

Array

  • A special variable that can store multiple values.
  • Each value has an index.
var varname=new Array("a","b","c");

Date Object

  • Used to manipulate and access the current date and time.
  • Created using the keyword new.

Math Object

  • Allows mathematical calculations.
  • Includes predefined methods and constants.
